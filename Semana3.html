SEMANA 3

1. VIERNES 05 DE ABRIL (presencial), continuamos explorando la plataforma Visual Studio Code, ampliando nuestro conocimiento sobre su uso y sus comandos. Dado nuestro objetivo de crear una bitácora digital a través de VSC, el profesor nos presentó una página web que ofrece una colección gratuita de plantillas HTML de alta calidad con licencia Creative Commons, llamada html5up. Esta herramienta nos facilitaría el proceso, por lo que descargamos un diseño de nuestra preferencia. Después, antes de actualizar la información en Visual Studio Code, procedimos a crear una cuenta en GitHub, creamos un repositorio y adjuntamos las carpetas del archivo de la plantilla que descargamos en html5up. De manera automática, se creó nuestra página web, lista para ser actualizada poco a poco.

2. SÁBADO 06 DE ABRIL (virtual), al inicio de la clase, repasamos cómo crear un repositorio en GitHub y lo vinculamos con Visual Studio Code. Posteriormente, instalamos un software de control de versiones, Git scm, que nos ayudará a sincronizar los avances realizados en Visual Studio Code con nuestra página web. Una vez instalado, clonamos el repositorio de GitHub en Visual Studio Code y realizamos algunos cambios para asegurarnos de que todo estuviera funcionando correctamente. Utilizando la función de control de código fuente, sincronizamos los cambios con GitHub para confirmar su correcta actualización. Finalmente, abordamos de manera breve la ideación y prototipado rápido, centrándonos en nuestro proyecto final. Para esto, el profesor nos asignó la tarea de filtrar nuestras ideas de soluciones, determinar su viabilidad y seleccionar una para conceptualizarla.
